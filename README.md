# EdmData Resources

A community repository for presentation materials, code samples, notebooks, datasets, and learning resources shared by presenters and members of the Edmonton Microsoft Data Professionals (Edmdata) community.

EdmData is a community for data, analytics, AI, and Microsoft data platform professionals in the Edmonton area. This repository serves as a central hub for event resources, helping attendees revisit presentations, explore demos, continue learning, and connect with the broader community.

## What You'll Find
* Presentation slides and supporting materials
* Demo code and sample projects
* Jupyter notebooks and tutorials
* Sample datasets (where permitted)
* Reference links and additional learning resources

## Contributing
Presenters and community members are welcome to contribute resources related to EDMData events. Please ensure that any shared content is publicly distributable and appropriately licensed.
